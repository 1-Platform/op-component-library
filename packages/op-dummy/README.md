# This is op-dummy 👋

Here is some information about op-dummy

## Install

```sh
npm install
```

## Usage

### Development server
- Install packages

```sh
npm run install
```

- Run development server

```sh
npm run dev
```

### Build

```sh
npm run build
```

## Run tests

```sh
npm run build
npm run test
```
